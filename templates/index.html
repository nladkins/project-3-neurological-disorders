Neurological Disorders Dashboard

Use the interactive charts below to explore the datasets



The prevalence of neurological disorders has expanded over the years, including diagnoses such as Autism, Attention Deficit and Hyperactivity Disorder(ADHD), intellectional disability, and Asperger's Syndrom. Not only can this have a debilitating effect on the individual, but this also affects the family members who are often providing the supports that are needed. The purpose of this website is to use datasets available from Our World in Data that is available on Neurological Disorder statistics across the globe. The project entailed cleaning data from the website that had a number of quality issues and pair it down to these four disorders listed below. The graphs use Chart.js and the site uses a Flask App on the backend. For more information on the code and data, please visit the GitHub repository below.
